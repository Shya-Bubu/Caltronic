# Exam Patterns — h-Parameter Extraction

## Common Question Types

### Type 1: Extract from Output Characteristics (5–8 marks)
"Given the output characteristics below, determine h_fe and h_oe at the Q-point (I_BQ = 40 μA, V_CEQ = 6 V)."

**Steps:** Draw vertical line at V_CEQ → read I_C at two adjacent I_B curves → h_fe = ΔI_C/ΔI_B. Read slope of one curve at Q → h_oe = ΔI_C/ΔV_CE.

### Type 2: Extract from Input Characteristics (5 marks)
"Using the input characteristics, find h_ie and h_re."

**Steps:** On V_CEQ curve, find slope ΔV_BE/ΔI_B at Q → h_ie. On horizontal I_BQ line, find ΔV_BE/ΔV_CE between two curves → h_re.

### Type 3: Comparison Across Configurations (3 marks)
"Explain why common base has a much lower input impedance than common emitter."

## Common Mistakes

1. **Reading the wrong slope.** For $h_{oe}$, use the slope of ONE curve; for $h_{fe}$, use the SPACING between curves.
2. **Wrong units.** $h_{oe}$ is in μA/V (or μS), not in kΩ. Convert if needed.
3. **Forgetting the Q-point constraint.** The values depend on WHERE on the curves you measure.

## Time Management
- Read two h-parameters from one graph: 3–4 minutes
- Complete set from both graphs: 6–8 minutes
