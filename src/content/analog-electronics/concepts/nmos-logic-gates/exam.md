# Exam Focus — NMOS NOR and NAND Gates

## What Examiners Test
NMOS gate questions are circuit analysis (4–8 marks) and truth table derivation (3–5 marks).

## Common Question Types

### Type 1: Draw the NMOS NOR/NAND Gate (4 marks)
NOR: parallel drivers + load. NAND: series drivers + load. Label all transistors, VDD, inputs, output.

### Type 2: Derive the Truth Table (3–4 marks)
For each input combination, determine which transistors are ON/OFF, trace current path, determine output.

### Type 3: Explain Parallel vs Series (3 marks)
Parallel → NOR (any ON pulls down, faster). Series → NAND (all must be ON, slower due to stacked R_on).

## Common Mistakes
- Confusing NOR and NAND (remember: parallel = NOR, series = NAND)
- Not recognising that the load provides the inversion
- Forgetting static power dissipation exists in NMOS logic
