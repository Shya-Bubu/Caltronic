# Exam Guide: h-Parameter Model

## Typical Questions
| Pattern | Marks | Time |
|---------|-------|------|
| **Draw full/simplified h-parameter model** | 4-5 | 3 min |
| **Find hfe, hoe from output characteristics** | 4-6 | 5 min |
| **Convert CE ↔ CC parameters** | 2-3 | 2 min |
| **Explain when to use simplified vs full** | 3-4 | 3 min |

## Common Mistakes
- **Confusing hfe (AC β) with hFE (DC β)**: hfe is the small-signal gain; hFE is the DC ratio IC/IB. They're similar but not identical.
- **Wrong direction for hfe current source**: hfe·ib flows from collector to emitter (into the collector terminal for NPN).
- **Forgetting 1/hoe is a resistance**: hoe has units of siemens (conductance). The resistance is 1/hoe, which goes in parallel with RC.
