# Exam Guide: Load Lines and Graphical Analysis

## Typical Questions
| Pattern | Marks | Time |
|---------|-------|------|
| **Draw DC and AC load lines** | 5-6 | 5 min |
| **Find x-intercepts of both lines** | 3-4 | 2 min |
| **Construct output waveform graphically** | 6-8 | 8 min |
| **Determine clipping limits** | 3-4 | 3 min |

## Common Mistakes
- **Using wrong R for each line**: DC uses RC+RE. AC uses RC||RL. Don't mix them up.
- **Not passing AC line through Q-point**: The AC load line must go through the Q-point, not the origin or VCC.
- **Reading wrong load line for waveform**: Use the AC load line for waveform construction, not the DC line.
