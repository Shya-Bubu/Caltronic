# Exam Guide: BJT Structure and Operation

## Typical Questions
| Pattern | Marks | Time |
|---------|-------|------|
| **Identify BJT type from symbol** | 1-2 | 1 min |
| **Calculate IC, IB, IE from β** | 3-4 | 2 min |
| **Determine operating region from junction biasing** | 2-3 | 2 min |
| **Explain why active region is used for amplification** | 3-4 | 3 min |

## Common Mistakes
- **Confusing β and α**: β = IC/IB (large, ~100). α = IC/IE (close to 1, ~0.99). They're related by α = β/(β+1).
- **Forgetting IE = IC + IB**: Some students use IE = IC. For β=100, the error is only 1%, but examiners check.
- **Wrong arrow direction**: NPN arrow points OUT. If you draw it pointing in, you've drawn a PNP.
