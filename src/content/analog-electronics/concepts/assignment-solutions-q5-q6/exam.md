# Exam Guide: Q5 & Q6 Solution Techniques

## Q5 (CE with Partial Degeneration)
- Re1 bypassed, Re2 stays → gain ≈ −Rout/Re2
- Include hoe when 1/hoe is comparable to Rc
- Rin,base = hie + (1+hfe)Re2 — degeneration increases Rin
- Overall: include source and load dividers

## Q6 (Two-Stage CE-CE + Feedback)
- Each stage independently: Thevenin → KVL → Q-point
- Two CE stages: overall gain positive (two inversions)
- Feedback topology: identify sampling (V/I at output) and mixing (series/shunt at input)
- Loading: Rf as passive element at both input and output
