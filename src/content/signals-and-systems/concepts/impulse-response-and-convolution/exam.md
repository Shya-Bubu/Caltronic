# Exam Focus — Impulse Response & Convolution

## What Examiners Test
This is the highest-value topic in the lesson (8–15 marks). Questions test both understanding and computation.

## Common Question Types

### Type 1: Compute Convolution (6–10 marks)
Given x(t) and h(t), compute y(t) = x(t)*h(t). Show the flip-shift-multiply-integrate steps explicitly.

### Type 2: Derive Convolution Formula (4–6 marks)
Show how the convolution integral follows from LTI properties (sampling → scaling → TI → additivity → limit).

### Type 3: Find Impulse Response from Step Response (3–4 marks)
Given s(t), find h(t) = ds/dt, or vice versa.

## Common Mistakes
- Forgetting to flip h(τ) to get h(t−τ) (not h(τ−t))
- Wrong integration limits (especially for causal signals)
- Confusing convolution (*) with multiplication (·)
- Not showing the derivation steps when asked to "derive"
