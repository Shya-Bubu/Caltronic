# Exam Focus — Initial & Final Value Theorems

## Common Question Types

### Type 1: Apply IVT/FVT to Given Y(s) (3–4 marks)
Given Y(s), find y(0⁺) and y(∞) using the theorems. State any validity conditions.

### Type 2: Verify Against Time-Domain Solution (2–3 marks)
Compute y(t) fully, then verify using IVT/FVT for consistency.

### Type 3: Determine if FVT is Applicable (2–3 marks)
Check pole locations of sY(s). FVT only works for stable signals.

## Common Mistakes
- Forgetting to multiply Y(s) by s before taking the limit
- Applying FVT to oscillating signals (purely imaginary poles)
- Confusing the direction of the limit: IVT → s→∞, FVT → s→0
