# Exam Focus — Ideal Filter Types

## Common Question Types

### Type 1: Sketch the Magnitude Response (4–6 marks)
Given a filter description in words, sketch $|H(\omega)|$ vs $\omega$. Mark cutoff frequencies, unity gain regions, and zero regions. Include both positive and negative $\omega$ — examiners penalise one-sided sketches.

### Type 2: Write the Mathematical Definition (4 marks)
Express $|H(\omega)|$ as a piecewise function. Be precise with inequalities (≤ vs <). Different questions may use strict or non-strict inequalities — match the given filter specification.

### Type 3: Identify Filter Type from Response (3 marks)
Given a magnitude plot or piecewise definition, name the filter type and identify the bandwidth parameter(s).

### Type 4: Cascading Filters (5–6 marks)
Two filters cascaded — find the overall magnitude response by multiplying $|H_1| \cdot |H_2|$. Draw the result. Key: if either filter has $|H| = 0$ at some $\omega$, the cascade also has $|H| = 0$ there.

## Common Mistakes
- Forgetting the **negative frequency** side (magnitude response is symmetric for real systems)
- Confusing BPF and BSF — remember: BPF passes the band, BSF stops the band
- Using wrong inequality direction at cutoff boundaries
- Not labelling cutoff frequencies on the sketch
