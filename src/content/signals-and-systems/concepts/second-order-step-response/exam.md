# Exam Focus — Second-Order Step Response

## Common Question Types

### Type 1: Classify Damping Case (4–6 marks)
Given R, L, C values, compute discriminant or compare R to 2√(L/C). State: overdamped, critically, or underdamped.

### Type 2: Find Poles and Sketch Response (6–8 marks)
Solve characteristic equation, identify pole types, sketch step response shape.

### Type 3: Apply IVT/FVT to Second-Order System (4–6 marks)
Use value theorems to verify initial and final values of step response.

## Common Mistakes
- Using wrong formula for R_crit (it's 2√(L/C), not 2√(LC))
- Forgetting that current output has y(∞)=0 while voltage output has y(∞)=1
- Not identifying the correct standard form coefficients
- Confusing ωₙ (natural frequency) with ω_d (damped frequency)
