# Exam Focus — System Interconnections

## Common Question Types

### Type 1: Find Overall Transfer Function (4–6 marks)
Given block diagram with series/parallel/feedback, derive H(s).

### Type 2: Draw Block Diagram (3–4 marks)
Given a system description, draw the series/parallel/feedback block diagram.

### Type 3: Closed-Loop Transfer Function (4–6 marks)
Derive G/(1+GF) from first principles using the error signal.

## Common Mistakes
- Forgetting the sign convention in feedback (negative vs positive)
- Not recognising that series LTI systems can be reordered
- Writing H₁+H₂ for series (it's H₁·H₂) or H₁·H₂ for parallel (it's H₁+H₂)
