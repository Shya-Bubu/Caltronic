# Exam Focus — System Classification & LTI

## What Examiners Test
System classification is a frequent 4–8 mark question type. You'll be given a system equation and asked to determine if it's linear and/or time-invariant.

## Common Question Types

### Type 1: Classify a Given System (4–6 marks)
Given $y(t) = f(x(t))$, test for linearity (superposition) and time invariance. Show both tests explicitly.

### Type 2: Definition Questions (2–3 marks)
State the additivity and scaling properties. Define time invariance mathematically.

### Type 3: Identify LTI from Multiple Systems (3–4 marks)
Given several system equations, identify which ones are LTI.

## Common Mistakes
- Testing only additivity and forgetting scaling (both needed for linearity)
- Confusing $y(t-\tau)$ with substituting $t-\tau$ into the system equation incorrectly
- Assuming all causal systems are LTI
- Forgetting that $y(t) = x(t) + 1$ is NOT linear (the constant term violates scaling)
