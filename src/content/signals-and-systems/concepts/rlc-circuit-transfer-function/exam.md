# Exam Focus — RLC Transfer Function

## Common Question Types

### Type 1: Derive H(s) for RLC Circuit (8–12 marks)
Write KVL in time domain, transform to s-domain, solve for H(s). Simplify to standard form. Show impedance method as verification.

### Type 2: Classify Damping (4–6 marks)
Given R, L, C values, determine if the system is overdamped, critically damped, or underdamped. Calculate ωₙ and ζ.

### Type 3: Find Poles (4–6 marks)
Solve the characteristic equation s² + (R/L)s + 1/(LC) = 0. Identify pole locations and system response type.

## Common Mistakes
- Forgetting to include the 1/C integral term for the capacitor in KVL
- Wrong sign in KVL (voltage drops are positive around the loop)
- Confusing the standard form parameters (ωₙ vs ω_d, ζ vs α)
- Not simplifying to standard second-order form before analysing
