# Exam Focus — Step Response of LTI Systems

## Common Question Types

### Type 1: Compute Step Response (8–12 marks)
Given H(s), find the step response: Y(s)=H(s)/s, partial fractions, inverse Laplace. Show every step.

### Type 2: Find Time Constant and Settling Time (3–4 marks)
Given R, C (or the transfer function), find τ=RC and settling time ≈5τ.

### Type 3: Sketch the Step Response (3–4 marks)
Draw y(t) vs t showing initial value, time constant marking at 63.2%, and final value.

## Common Mistakes
- Forgetting the u(t) multiplier (causal signal — zero for t<0)
- Wrong partial fraction coefficients (cover-up method errors)
- Confusing the step response with the impulse response
- Not simplifying H(s)·(1/s) before doing PFE
