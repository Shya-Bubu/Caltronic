# Exam Focus — Transfer Function & RC Circuit

## Common Question Types

### Type 1: Derive H(s) for a Given Circuit (6–10 marks)
Apply KVL/KCL in time domain, take Laplace transform (zero ICs), extract H(s)=Y(s)/X(s). Show every step.

### Type 2: Impedance Method (4–6 marks)
Use voltage divider with s-domain impedances. Faster but may carry fewer marks.

### Type 3: Identify System Properties from H(s) (4–6 marks)
Find poles, DC gain, time constant, stability from H(s). Sketch magnitude response.

## Common Mistakes
- Forgetting to assume zero initial conditions (capacitor uncharged, no initial current)
- Using 1/C instead of 1/Cs for capacitor impedance
- Wrong voltage divider formula (numerator vs denominator impedance)
- Not simplifying H(s) to standard form
