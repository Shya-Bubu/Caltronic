# Exam Focus - Detecting jω-Axis Poles from the Routh Table

## What the question usually wants

This topic is tested as a Routh-table classification problem with a special zero entry. A typical answer is expected to:

- build the array correctly
- identify the zero in the first column
- explain what that zero means
- conclude whether the system has a pair of poles on the $j\omega$ axis

## Best answer structure

1. Write the polynomial in descending powers of $s$.
2. Build the Routh array row by row.
3. Mark the first-column entries clearly.
4. If a zero appears, state how the note handles it.
5. Write the final conclusion about poles on the $j\omega$ axis.

## Common mistakes

- treating the zero as an arithmetic accident and moving on
- counting sign changes without discussing the special case
- forgetting that the note singles out the row indexed by $s^1$
- writing "unstable" immediately without checking the actual first-column pattern

## One sentence worth remembering

If all first-column entries are positive except a zero in the row indexed by $s^1$, the note identifies one pair of poles on the $j\omega$ axis.
