# Exam Focus — Laplace Domain Analysis

## Common Question Types

### Type 1: Derive Y(s..) = H(s)X(s) (4–6 marks)
Show the convolution theorem derivation by swapping integration order and identifying H(s) and X(s).

### Type 2: Find Output in S-Domain (4–6 marks)
Given H(s) and X(s), find Y(s) by multiplication, then inverse transform.

### Type 3: State the Convolution Theorem (2–3 marks)
"Convolution in time ↔ multiplication in s-domain."

## Common Mistakes
- Forgetting that this only applies to LTI systems
- Confusing convolution (*) with multiplication (·) in the time domain
- Not justifying the swap of integration order (Fubini's theorem)
